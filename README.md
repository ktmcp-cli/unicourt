# @ktmcp/unicourt

MCP server for UniCourt Enterprise APIs

## Installation

```bash
npm install -g @ktmcp/unicourt
```

## Usage

Add to your MCP settings:

```json
{
  "mcpServers": {
    "unicourt": {
      "command": "mcp-unicourt"
    }
  }
}
```
